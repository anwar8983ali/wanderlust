# 🚀 Trading Dashboard App

A full-stack trading dashboard with BUY/SELL functionality, holdings tracking, and order management.

---

## 🌐 Live Demo

🔗  [Live App Link](https://wanderlust-1-hsi8.onrender.com)  
> ⚠️ Note: Backend may take a few seconds to wake up (Render free tier)

---

## 🛠 Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Deployment: Vercel (Frontend), Render (Backend)

---

## ✨ Features

- ✅ Buy & Sell Stocks  
- ✅ Real-time Holdings Update  
- ✅ Orders History  
- ✅ Portfolio Summary (P&L)  
- ✅ Interactive UI
